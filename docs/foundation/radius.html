Radius

The DIGIT Radius System defines corner radius values applied consistently across all components. Radius influence how an interface feels. They soften the visual weight of elements and help create a sense of order and harmony.

Radius System

The DIGIT Radius System provides 14 predefined radius levels, each corresponding to a consistent corner curvature value.
